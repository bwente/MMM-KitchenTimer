# Changelog

All notable changes to MMM-KitchenTimer are documented here.

## [Unreleased]

- Added repository maintenance and catalog metadata.

## [1.0.1] - 2026-07-22

- Kept active timers and completion events running while the module is hidden.

## [1.0.0] - 2026-07-22

- Modernized the original timer with reliable wall-clock timing.
- Added touch-friendly presets, pause/resume, and reset/dismiss controls.
- Added compact and full-screen layouts.
- Added incoming control notifications and outgoing semantic state notifications.
- Added automated tests with Node's built-in test runner.
